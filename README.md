# scRNA-seq Cell Classification

## Overview
This repository contains a Deep Learning project aimed at classifying different cell types based on gene expression profiles. The primary goal of this project is to explore model behaviors, specifically identifying when models underfit or overfit, and effectively applying techniques to mitigate overfitting.

## Project Structure
- `ADI_scRNA_classification_Santiago_López.ipynb`: The main Jupyter Notebook containing the data exploration, preprocessing, model implementation, and evaluation.
- `HW1DL_Idoia/`: Directory containing project specific files. *(Note: This folder is excluded from version control for privacy reasons)*.

## Objectives
- Train deep learning models to classify cell types accurately.
- Observe and analyze scenarios where models **underfit** and **overfit** the data.
- Implement regularization techniques (e.g., Dropout, L1/L2 regularization, Early Stopping) to address and resolve overfitting problems.
